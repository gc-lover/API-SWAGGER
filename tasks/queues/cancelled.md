# Очередь API задач — статус `cancelled`

- Лимит файла: ≤ 500 строк. При превышении создайте `cancelled_0001.md`, `cancelled_0002.md`.
- Формат записи:

```markdown
- **Task ID:** API-TASK-175  
  **Источник:** .BRAIN/02-gameplay/economy/economy-currency-exchange.md (v1.0.0)  
  **Отменено:** 2025-11-09 00:32 — ДУАПИТАСК  
  **Причина:** Концепция пересматривается, повторно создадим после обновления документа.
```

- При повторном запуске задачи удалите запись отсюда и заведите новую в актуальном файле статуса.
