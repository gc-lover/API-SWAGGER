# Очередь API задач — статус `review`

- Лимит файла: ≤ 500 строк. При превышении создайте `review_0001.md`, `review_0002.md`.
- Формат записи:

```markdown
- **Task ID:** API-TASK-206  
  **Спецификация:** api/v1/gameplay/combat/combat-session.yaml  
  **Отправлено на ревью:** 2025-11-09 00:20 — АПИТАСК  
  **Что проверить:** Описания эвентов и security схемы.
```

- После ревью переместите запись в соответствующий файл (`completed.md`, `needs-work` → верните в `in-progress.md`).
