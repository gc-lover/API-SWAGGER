# Очередь API задач — статус `archived`

- Лимит файла: ≤ 500 строк. При превышении создайте `archived_0001.md`, `archived_0002.md`.
- Формат записи:

```markdown
- **Task ID:** API-TASK-150  
  **Спецификация:** api/v1/gameplay/world/travel-events.yaml  
  **Архивировано:** 2025-11-09 00:30 — ДУАПИТАСК  
  **Причина:** Backend и Frontend завершены, Clear Agent подтвердил закрытие.
```

- Сохраняйте записи для истории. При обнаружении ошибок верните задачу в нужный статус и удалите запись из архива.
