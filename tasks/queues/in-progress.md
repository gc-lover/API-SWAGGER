- Лимит файла: ≤ 500 строк. При превышении создайте `in-progress_0001.md`, `in-progress_0002.md`.
- Формат записи:

```markdown
- **Task ID:** API-TASK-204  
  **Источник:** .BRAIN/05-technical/backend/inventory-system/part1-core-system.md (v1.0.1)  
  **Взято в работу:** 2025-11-09 00:12 — API Executor  
  **Target:** api/v1/inventory/inventory-core.yaml
```

- После завершения этапа перенесите запись в файл следующего статуса (`ready-for-spec.md`, `completed.md` и т.д.).
