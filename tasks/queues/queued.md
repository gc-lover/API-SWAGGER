# Очередь API задач — статус `queued`

- Лимит файла: ≤ 500 строк. При превышении создайте `queued_0001.md`, `queued_0002.md`.
- Формат записи:

```markdown
- **Task ID:** API-TASK-201  
  **Источник:** .BRAIN/05-technical/backend/auth/README.md (v1.0.1)  
  **Создано:** 2025-11-09 00:08 — ДУАПИТАСК  
  **Target:** api/v1/auth/authentication.yaml
```

- При смене статуса удалите запись отсюда и перенесите в файл нового статуса (например, `in-progress.md`).
